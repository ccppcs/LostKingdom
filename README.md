# Lost kingdom은 MFC로 온라인 2D ARPG 입니다.

   * 방을 만들고 접속을 합니다.
   * 채팅을 통해 서로 선택할 캐릭터를 정하고 게임을 시작합니다.
   * 게임 중 채팅을 할수 있어 전략적 플레이가 가능합니다.
   * 몬스터 처치시 골드와 경험치를 얻을수 있으며, 스킬 레벨업, 상태 강화를 할수 있습니다.
   * 모든 몬스터를 처치하면 다음 스테이지로 넘어가며, 전원 사망시 게임이 종료됩니다.
